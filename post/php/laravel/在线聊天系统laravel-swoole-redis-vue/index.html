在线聊天系统(laravel swoole redis vue)

一.前端页面完成

1.安装准备

安装前必须保证系统已经安装了下列软件

PHP >= 7.1.3

PHP OpenSSL 扩展

PHP PDO 扩展

PHP Mbstring 扩展

PHP Tokenizer 扩展

PHP XML 扩展

PHP Ctype 扩展

PHP JSON 扩展

PHP BCMath 扩展

2.下载安装node.js

http://nodejs.cn/download/

3.laravel初始化前端依赖

npm install

4.安装 Vue 弹幕组件 vue-baberrage

npm install vue-baberrage –save

5.引入 vue-baberrage

在 resources/js/app.js 中引入 vue-baberrage 组件

window.Vue = require(‘vue’);

import {vueBaberrage} from ‘vue-baberrage’;

Vue.use(vueBaberrage);
