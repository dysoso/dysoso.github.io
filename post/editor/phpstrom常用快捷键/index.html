CTRL+/ 单行注释/取消注释

CTRL+SHIFT+/ 块状注释/取消块状注释

Ctrl+Shift+U 选中的字符大小写转换

CTRL+ALT←/→ 返回上次编辑的位置

ALT+←/→ 切换代码视图，标签切换

ALT+↑/↓ 在方法间快速移动定位

CTRL+SHIFT+[] 直接选中块代码 = CTRL+W 按好几下

CTRL+ [] 光标移动到{}[]开头或结尾位置

CTRL+ALT+F12 资源管理器打开文件夹，跳转至当前文件在磁盘上的位置

CTRL+Z 倒退(代码后悔)

CTRL+SHIFT+Z 向前

SHIFT+F2 高亮错误或警告快速定位错误，多个错误循环高亮

CTRL+C 复制

CTRL+V 粘贴

CTRL+X 剪切,删除行

Ctrl + Y 删除行插入符号

CTRL+D 复制行 , 快速分布li标签等

Ctrl + Shift + [+/-]  折叠/展开全部代码

Ctrl + [+/-]  折叠/展开代码

Ctrl+Shift+J快捷键，所有多余的字符(空格，引号和加号)被去掉了

CTRL+J 自动代码提示，自动补全

alt +'7':显示当前的类/函数结构。类似于eclipse中的outline的效果。试验了一下，要比aptana的给力一些，但还是不能完全显示prototype下面的方法名。

Ctrl + Alt + L	普通的等号赋值都可以对齐，数组的键值对也可以对齐。

ALT + J 选定多个文本
